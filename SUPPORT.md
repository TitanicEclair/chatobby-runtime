# Support

Use the
[Chatobby connector issue tracker](https://github.com/TitanicEclair/chatobby-obsidian/issues)
for installation defects, reproducible runtime failures, and feature requests.
Include the connector version, runtime version, Windows version, concise steps,
and the exact visible error after removing private content.

Security issues follow [SECURITY.md](SECURITY.md). General questions may be
sent to `thatsmad002@gmail.com`.

Chatobby is free during public alpha. Optional
[Patreon support](https://www.patreon.com/cw/MadelynCruzTan/membership) is
appreciated but does not purchase support priority or unlock functionality.
